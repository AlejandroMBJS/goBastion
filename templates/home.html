Get Started in Seconds

Start building your next project with goBastion

$ go install goBastion
$ go-bastion init myapp

Install and initialize your project

$ cd myapp
$ go-bastion serve

Start the development server

$ go-bastion migrate
$ go-bastion create-admin

Run migrations and create admin user
